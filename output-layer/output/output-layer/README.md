# output-layer 产物目录

每个主题一个文件夹，**打开就是最新那份成品**。历史版本在 `_archive/`，要时才翻。

> 此目录由 `scripts/organize.py` 维护。生成新产物后跑 `python3 organize.py --apply` 即可重新整理。

## 目录结构

```
output-layer/
  README.md            ← 本文件
  <主题>/              ← 最新成品（output.docx / output.clean.md 等）
  _archive/<主题>/<时间戳>/   ← 历史份
```
